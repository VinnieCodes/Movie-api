The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

/users Post { "id": 1, "name": "Anaiah", "favoriteMovies": [ "Shrek", "Intersteller" ] } { "id": 1, "name": "Anaiah", "favoriteMovies": [ "Shrek", "Intersteller" ] }
/users/:id put none A json object containing the details of the movie watcher { "id": 1, "name": "Anaiah", "favoriteMovies": [ "Shrek", "Intersteller" ] }
/users/:id/:movieTitle post {"name": "bob", "favoriteMovies": []} A message saying shrek has been added to user 1's array
/users/:id/:movieTitle delete none A message saying `${movieTitle} has been removed from user ${id}'s array`
/users/:id delete none deletes a users and displays message `user ${id} has been deleted`
/movies get none A list of all the detail of all the movies
/movies/:title get none details about the movie that was put in the URL
/movies/genre/:genreName get none All the movies that fit that genre
/movies/directors/:directorName put get shows details about the director
